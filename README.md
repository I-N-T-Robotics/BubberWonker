# RamHorns
Library for Motion Profiling

### How to use

Add it in your root build.gradle at the end of repositories:

```gradle
allprojects {
  repositories {
    maven { 
      url 'https://jitpack.io'
    }
  }
}
```
Add the dependency

```gradle
dependencies {
  implementation 'com.github.frc4571:RamHorns:-SNAPSHOT'
}
```
